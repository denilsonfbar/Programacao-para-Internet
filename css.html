Propriedade display

A propriedade display especifica como um elemento deve ser mostrado. Todo elemento tem um valor para display, normalmente block, inline ou inline-block.

Elementos block forçam uma quebra de linha após sua exibição e permitem a definição das propriedades width e height. Se a largura não for definida, o elemento se estende até o final do container. Se a altura não for definida, ela se ajusta automaticamente ao conteúdo.

Propriedade overflow

Controla o que acontece quando o conteúdo é maior que as dimensões definidas para o elemento.

overflow:visible permite que o conteúdo ultrapasse as dimensões. Valor default.

overflow:hidden oculta o overflow.

overflow:scroll oculta o overflow e exibe uma barra de rolagem.

overflow:auto adiciona barras de rolagem somente quando necessário.

Propriedade position

Um elemento com position:static é posicionado de acordo com o fluxo normal da página:

position:static

Um elemento com position:relative é posicionado em relação à sua posição normal, alterando-se as propriedades top, right, bottom, left:

position:relative

Um elemento com position:fixed é posicionado em relação ao viewport, definindo-se as propriedades top, right, bottom, left.

position:fixed

Um elemento com position:absolute é semelhante à position:fixed, mas o elemento é fixado em relação ao elemento ancestral mais próximo com posição definida.

position:relative
position:fixed

Um elemento com position:sticky alterna entre relative e fixed dependendo da posição da barra de rolagem.

Normalmente utilizada para posicionar uma imagem em relação ao texto em um container.

Os valores possíveis são left, right, none e inherit.
